# バトル画面 設計書

デジモンカードゲーム公式ルール準拠。チュートリアルモード＋対戦モード共通の基盤。

---

## フィールド構成

```
【相手側（AI）】
[デッキ] [バトルエリア × 5スロット] [セキュリティ]
                                      [育成エリア]

[============= メモリーゲージ(-10 〜 +10) =============]

【自分側（プレイヤー）】
[セキュリティ]                         [デッキ]
[育成エリア]  [バトルエリア（無制限/横スクロール）] [トラッシュ]

[フェーズバー：現在フェーズ / 手札数 / デジタマ数 / トラッシュ数]
[手札エリア（カード重ねる）]
[アクションバー：ターン終了 / ゲート退出する]
```

---

## ゲーム開始

1. デッキをシャッフル
2. デジタマデッキ（Lv.2）とメインデッキを分離
3. 手札5枚をドロー
4. マリガン判定（1回のみ引き直し可能）
5. セキュリティを5枚セット（デッキ上から）
6. 先攻プレイヤーからターン開始（メモリー0）

---

## ターンの流れ

### 1. アクティブフェイズ（自動処理）
- レスト状態（横向き）のデジモンをすべてアクティブ（縦向き）に戻す（例外除く）
- **演出**: battle-controls参照
- **例外**: 効果でアクティブにならない付与されているカードはレスト状態のまま

### 2. ドローフェイズ（自動処理）
- デッキからカードを1枚引く
- **例外**: 先攻1ターン目はドローなし
- **敗北条件**: デッキ0枚でドローフェイズを迎えたら敗北
- **演出**: battle-controls参照

### 3. 育成フェイズ（プレイヤー操作）
以下のどれか1つを選択。操作完了後、自動でメインフェイズへ移行。
- **例外**: デジタマ0枚の時→強制スキップ（育成エリアのカード裏面画像表示を削除）

#### A. デジタマを孵化する（タップ/クリック）
- **条件**: 育成エリアが空 かつ デジタマデッキに1枚以上ある
- **操作**: デジタマデッキをタップ
- **処理**: デジタマデッキの上から1枚を育成エリアに置く
- **演出**: battle-controls参照

#### B. 育成エリアのデジモンをバトルエリアへ移動（長押し＋スワイプ）
- **条件**: 育成エリアにLv.3以上のデジモンがいる（Lv.2デジタマは移動不可）
- **操作**: 育成エリアのカードを長押し（0.4s）→ スワイプ
- **処理**: バトルエリアの空きスロットに移動。メモリーコスト不要
- **演出**: battle-controls参照

#### C. 何もしない（ボタン）
- 「何もしない → メインフェイズへ」ボタンをタップ

### 4. メインフェイズ（プレイヤー操作）
以下のアクションを自由に行える（順不同、複数回可能）。
メモリーが1以下になった場合、相手にターンが移る。

#### 登場（プレイ）
- **条件**: 手札にデジモン/テイマー/オプションカードがある。
- **操作**: 手札カードをドラッグ → バトルエリアの空きスロットにドロップ
- **コスト**: カードの「登場コスト」分だけメモリーを消費
- **演出**: battle-controls参照
- **登場時効果**: 【登場時】効果があれば、登場演出後に効果確認ダイアログ

#### 進化
- **条件**: 手札の進化カードの「進化条件」を満たすデジモンがバトルエリアまたは育成エリアにいる
- **操作**: 手札カードをドラッグ → 対象デジモンの上にドロップ
- **コスト**: カードの「進化コスト」分だけメモリーを消費（登場コストより安い）
- **処理**: 進化元カードはスタック（進化元）として下に重なる。進化元効果は有効
- **演出**: battle-controls参照
- **進化時効果**: 【進化時】効果があれば、進化演出後に効果確認ダイアログ

#### アタック
- **条件**: アクティブ状態（縦向き）のデジモンがバトルエリアにいる
- **操作**: デジモンを長押し（左スワイプ）して選択 → 「⚔ アタック」ボタン
- **処理**:
  1. アタック宣言 → デジモンがレスト状態（横向き）になる
  2. 相手のバトルエリアにデジモンがいる場合 → 相手はブロック宣言可能
  3. ブロックされた場合 → DPを比較。低い方が消滅
  4. ブロックされなかった場合 → セキュリティチェック
  5. セキュリティチェック: 相手のセキュリティ上から1枚めくる
     - セキュリティデジモン → DPを比較。アタック側のDPが低いとアタック側消滅
     - セキュリティオプション → 【セキュリティ】効果発動
     - チェック後、セキュリティカードはトラッシュへ
  6. 相手のセキュリティが0枚の状態でアタック → ダイレクトアタック → 勝利！
- **演出**:
  - battle-controls参照
  - セキュリティチェック: VS画面表示（セキュリティからめくれたカードはDPが相手より高くてもトラッシュへ行く※例外除く）
  　【例外】セキュリティ効果でトラッシュ以外の場所（バトルエリアや手札　等）にいく場合もあり。
  - 消滅: カードが砕けるエフェクト + トラッシュへ飛ぶ
  - ダイレクトアタック: VS画面表示 + 「DIRECT ATTACK!」テキスト + 右側にアタック中のデジモン、左側にプレイヤー画像とユーザー名

#### ブロック
- **条件**: 相手のアタック時、アクティブ状態の「ブロッカー」を持っているデジモンがバトルエリアにいる
- **操作**: ブロッカーが1体の場合：ブロックしますか？→はいorいいえ
　　　　　　 ブロッカーが2体以上いる場合：ブロックしますか？→はいorいいえ → ブロックするデジモン選択
- **処理**: ブロックするデジモンがレスト → DPを比較。低い方が消滅
- **演出**: battle-controls参照

### 5. ターン終了
- 「⏭ ターン終了」ボタンで手動終了
- またはメモリーが1以下になったら自動でターン移行
- メモリーの絶対値が相手側（右側）のメモリーになる

---

## メモリーシステム

- ゲージは -10 〜 +10
- 正の値 = プレイヤーのメモリー
- 負の値 = 相手のメモリー（ターンが移る）
- カードの登場/進化コスト分だけメモリーを消費
- メモリーが1以下になった瞬間、ターンが相手に移る
- 相手にターンが移る際、メモリーの絶対値が相手の開始メモリーになる
  - 例: メモリー-3で終了 → 相手のターン開始時メモリー3

---

## 消滅処理

- DPバトルで負けた、または効果で消滅したカード
- 進化元スタックごとすべてトラッシュへ
- 【消滅時】効果があれば発動
- **演出**: カードが赤く光る → 砕けてトラッシュへ飛ぶ（0.5s）

## カード操作まとめ（ホバープレビュー廃止）

| 対象 | 操作 | 結果 |
|------|------|------|
| 自分の手札 | タップ | カード詳細画面表示 |
| 自分の手札 | ドラッグ | バトルエリアへ登場 or 進化 |
| 自分のバトルエリア | タップ | カード詳細画面表示 |
| 相手のバトルエリア | タップ | カード詳細画面表示 |
| 育成エリア（育成フェーズ） | タップ | 孵化（デジタマデッキから） |
| 育成エリア（育成フェーズ） | 長押し+スワイプ | バトルエリアへ移動 |
| 育成エリア（全フェーズ） | タップ | カード詳細 |
| カード詳細画面 | - | 拡大画像 + 効果テキスト + 進化元効果（セキュリティ効果がある場合は、セキュリティ効果も表示） |

## 演出一覧

| タイミング | 演出 | 所要時間 |
|-----------|------|---------|
| ターン開始 | 大きいテキスト + パルスアニメ | 1.5s |
| フェーズ開始 | フェーズ名大きく表示 | 1.5s |
| デジタマ孵化 | 卵→カード出現 + 発光 | 0.8s |
| 登場 | DIGITAL APPEAR + フラッシュ | 2s |
| 進化 | DIGITAL EVOLUTION + 回転+フラッシュ | 2s |
| アタック宣言 | カード前進 + 衝撃波 | 0.5s |
| レスト | 90度回転 | 0.3s |
| 消滅 | 赤発光 + 砕けてトラッシュへ | 0.5s |
| セキュリティチェック | 左に自分のデジモン、右にセキュリティカード | 0.8s |
| ダイレクトアタック | 左に自分のデジモン、右にユーザー画像+ユーザー名 + 全画面フラッシュ + テキスト | 1.5s |
| 効果発動 | ⚡エフェクト + テキスト表示 | 1s |

---

## 勝敗条件

- **勝利**: 相手のセキュリティが0枚の状態でダイレクトアタック成功
- **敗北**: 自分のセキュリティが0枚の状態でダイレクトアタックを受ける
- **敗北**: デッキが0枚の状態でドローフェイズを迎える

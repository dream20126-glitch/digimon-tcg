# チュートリアルシステム 設計書

## 概要
チュートリアルはスプレッドシートのシナリオデータに基づいて進行する。
コード変更なしで、管理画面からチュートリアルの追加・編集・削除が可能。

---

## データ構造（スプレッドシート）

### シート名: 「チュートリアルシナリオ」

| カラム | 説明 | 例 |
|--------|------|-----|
| チュートリアルID | 紐づくチュートリアルデッキのID | beginner-01 |
| ステップ番号 | 実行順（小さい順） | 1, 2, 3... |
| フェーズ | どのフェーズで発動するか | start, breed, main, attack, endTurn |
| アクション | 何をするか | announce, guide, forceAction, wait, endBattle |
| メッセージ | 表示するテキスト | デジタマを孵化してみよう！ |
| ターゲット | ガイド矢印の対象 | iku-slot, hand-lv3, battle-slot-0 |
| 矢印位置 | 矢印の表示位置 | above, below, left, right |
| 条件 | 次に進む条件 | action:hatch, action:evolve, action:attack, auto |
| 自動終了 | このステップでバトル終了するか | true / false |
| AI行動 | AIの固定行動（空なら通常AI） | play:ST1-02, attack, pass |
| 制限 | プレイヤーの操作制限 | onlyHatch, onlyEvolve, noAttack, free |

---

## アクション種別

### announce（演出テキスト表示）
画面中央に大きいテキストを表示。1.5秒後に自動で次へ。
```
アクション: announce
メッセージ: 【先攻】YOUR TURN!!
```

### guide（ガイド矢印＋メッセージ）
対象要素に矢印とメッセージを表示。条件を満たしたら次へ。
```
アクション: guide
メッセージ: デジタマを孵化してみよう！
ターゲット: iku-slot
矢印位置: above
条件: action:hatch
```

### forceAction（強制アクション）
特定のアクションを自動実行する。AIの行動に使う。
```
アクション: forceAction
AI行動: play:ST1-02
```

### dialog（説明ダイアログ）
OKボタン付きの説明を表示。
```
アクション: dialog
メッセージ: アクティブフェイズでは、レストしたカードを縦に戻します
```

### wait（条件待ち）
プレイヤーの操作を待つ。条件を満たしたら次へ。
```
アクション: wait
条件: action:attack
制限: onlyAttack
```

### endBattle（バトル終了）
チュートリアルを終了して結果画面へ。
```
アクション: endBattle
メッセージ: チュートリアルクリア！おめでとう！
```

## チュートリアル例: 初級編①

目的: 各フェーズの流れを体験する（1ターンで終了）

| # | フェーズ | アクション | メッセージ | ターゲット | 条件 |
|---|---------|-----------|-----------|-----------|------|
| 1 | start | announce | 【先攻】YOUR TURN!! | - | auto |
| 2 | unsuspend | dialog | アクティブフェイズ：レスト状態のカードを縦に戻します | - | auto |
| 3 | draw | dialog | ドローフェイズ：先攻1ターン目はドローなし | - | auto |
| 4 | breed | guide | デジタマを孵化してみよう！タップしてね | iku-slot | action:hatch |
| 5 | main | dialog | メインフェイズ：カードをプレイ・進化・アタックができます | - | auto |
| 6 | main | guide | デジタマに手札を重ねて進化させよう！ | hand-lv3 | action:evolve |
| 7 | main | announce | ターン終了！ | - | auto |
| 8 | endTurn | forceAction | - | - | AI行動: play:ST1-02 |
| 9 | endTurn | endBattle | 初級編①クリア！フェーズの流れは覚えたかな？ | - | auto |

## チュートリアルエンジンの処理フロー

```
1. チュートリアルデッキ選択時、紐づくシナリオデータをGAS APIで取得
2. バトル開始時、シナリオをステップ順にソート
3. 現在のステップ番号を保持（currentStep = 0）
4. フェーズ遷移時 or プレイヤーアクション時にチェック:
   - 現在のステップのフェーズ＆条件が合致 → アクション実行
   - アクション完了 → currentStep++
   - 次のステップが auto → 即実行
   - endBattle → バトル終了画面へ
5. 制限が設定されていれば、該当以外の操作を無効化
6. AI行動が指定されていれば、通常AIの代わりにスクリプト行動を実行
```

## 管理画面の機能

### チュートリアルシナリオ設定画面
- チュートリアル一覧（既存のチュートリアルデッキに紐づけ）
- シナリオステップの追加・編集・削除
- ステップの並び替え（ドラッグ or 上下ボタン）
- 各ステップのフォーム:
  - フェーズ選択（プルダウン）
  - アクション選択（プルダウン）
  - メッセージ入力（テキスト）
  - ターゲット選択（プルダウン）
  - 条件選択（プルダウン）
  - AI行動入力（テキスト）
  - 制限選択（プルダウン）
- プレビュー機能（将来的に）
